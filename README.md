# Melomaniac

A music genre exploration app built with **SwiftUI**.

## Overview

Melomaniac is an iOS application that helps users explore and discover music genres. The app features an intuitive interface for browsing different music categories and learning about each genre.

## Features

- 🎵 **Multiple Music Genres**: Browse through various music genres including Classical, Hip-hop, Jazz, Pop, R&B, and Rock
- 📱 **SwiftUI Interface**: Modern, responsive user interface built entirely with SwiftUI
- 🎨 **Visual Genre Cards**: Beautiful genre cards with associated imagery
- 💫 **Smooth Navigation**: Seamless navigation between different sections of the app

Learning Resources
This project is based on Paul Hudson's 100 Days of SwiftUI, which is an excellent resource for learning SwiftUI development.
